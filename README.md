# HypeCheck AI

An AI-powered platform that analyzes short-form content to evaluate performance and guide creators with actionable insights.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
